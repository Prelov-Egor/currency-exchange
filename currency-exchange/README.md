Сервис обмена валют (НБ РБ)

Веб-сервис для двухэтапного обмена валют по официальным курсам Национального банка Республики Беларусь.

Основные возможности:



Автоматическое ежедневное обновление курсов с API НБ РБ

Двухэтапный обмен валют (котировка → подтверждение/отклонение)

Отчёт по сделкам за любой период

Список незавершённых сделок

История всех сделок

Удобный современный веб-интерфейс



Как запустить

Самый простой способ (локально):

python run.py

Через Docker (рекомендуется):

docker-compose up --build

Сервер будет доступен по адресу: http://localhost:8000

Основные возможности API



GET / — главная страница с интерфейсом

GET /api/v1/currencies — список доступных валют

POST /api/v1/exchange/quote — получение котировки обмена

POST /api/v1/exchange/confirm — подтверждение или отклонение сделки

GET /api/v1/deals/pending — список незавершённых сделок

GET /api/v1/deals/history — история всех сделок

GET /api/v1/reports/deals — отчёт по сделкам за период



Документация Swagger: http://localhost:8000/docs

Технологии



FastAPI + Uvicorn

SQLAlchemy (async) + SQLite

APScheduler (ежедневное обновление курсов)

Bootstrap 5 + JavaScript (фронтенд)

Ruff + Black (линтер и форматирование)
